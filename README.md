# InstaGridPoster📷
Simple console app for cut image to 3x3 equal parts

**⚠️Warning:HEIC format photos are not supported**⚠️
---

## 📦 Installation & Setup

### Requirements
- .NET 8+ 
- FFmpeg 8.0+
- ffmpeg.exe must be in the PATH environment variable

### Installation
```bash
git clone https://github.com/bulldogBoy/InstaGridPoster
cd InstaGridPoster
dotnet build
